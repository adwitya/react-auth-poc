# React Authentication POC

This project demonstrates a React Authentication Flow using Context API, Axios interceptors, and JWT token handling.

## Features
- Manual login with system password
- JWT token storage and validation
- Axios interceptors for API calls

## Setup
```bash
cd react-auth-poc
npm install
npm start
```

Update API URL in `src/api.js` and login endpoint in `src/AppWrapper.js`.
